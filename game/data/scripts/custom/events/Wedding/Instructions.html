Andromeda:
In order to get married you first have to be engaged.
To get engaged simply target your desired partner and type in the command ".engage" without the quotes.
For this to work, you both need to be on each other's friends list.
To do this, you need to type "/friendinvite PlayerName" without the quotes where PlayerName is the name of your partner.
Once engaged, you ask me to get married.
You both have to be wearing Formal Wear in order for this to work as well as the wedding will cost %fee% Adena.
Once married, you can use the command ".gotolove" without the quotes, to teleport to your partner at anytime.
If you wish to divorce, simply type the command ".divorce" without the quotes.
